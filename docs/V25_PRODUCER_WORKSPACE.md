# Mind Frontier Studio v25 — Producer Workspace

v25 unifies Atlas agents, memory, publishing intelligence, and prediction into one workspace.

## Workspace sections

- Research
- Producer outline
- Thumbnail recommendation
- Publishing recommendation
- Prediction
- Atlas Memory evidence
- Readiness score
- Warnings
- Saved workspace history

## API

- `POST /api/workspace/build`
- `POST /api/workspace/save`
- `GET /api/workspace/list`
- `GET /api/workspace/{workspace_id}`
