# Mind Frontier Studio v23 — Atlas Memory

v23 introduces a persistent memory graph.

## Entities

- Topics
- Categories

## Relations

- Topic belongs to category

## Stored evidence

- Project count
- Published count
- Average views
- Average retention
- Average subscribers gained
- Best project
- Confidence score

## API

- `POST /api/memory/rebuild`
- `GET /api/memory/overview`
- `GET /api/memory/search?q=...`
